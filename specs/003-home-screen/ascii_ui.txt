ホーム画面 ASCII UI

ホーム画面の目的:
- 最初に開いたときに今日やることが一目で分かる
- 復習開始、カード一覧、学習カード登録へ 1 アクションで移動できる
- 直近の学習状況と次に見るべき情報を同じ画面で把握できる

標準状態:
+----------------------------------------------------------------------------------+
| Forgetting Curve / 学習カードアプリ                                              |
+----------------------------------------------------------------------------------+
| [ホーム] [学習カード登録] [カード一覧] [復習] [統計] [設定]                      |
+----------------------------------------------------------------------------------+
| パンくず: ホーム                                                                 |
+----------------------------------------------------------------------------------+
| 今日の学習サマリ                                                                  |
| +----------------------------------+  +----------------------------------------+ |
| | 今日の復習対象: 12 枚          |  | 連続学習日数: 5 日                   | |
| | 期限超過: 3 枚                  |  |                                      | |
| | 未学習: 4 枚                    |  |                                      | |
| | [復習を始める]                  |  | [統計を見る]                          | |
| +----------------------------------+  +----------------------------------------+ |
+----------------------------------------------------------------------------------+
| クイックアクション                                                                |
| [今日の復習を始める]  [カード一覧を見る]  [学習カードを追加]  [設定を開く]       |

| 共通フッター                                                                      |

状態バリエーション:

1) 初回利用 / データなし
+----------------------------------------------------------------------------------+
| ようこそ                                                                          |
| まだ学習カードがありません                                                        |
| 最初の 1 枚を登録すると、ここに学習状況と今日の予定が表示されます                 |
| [最初のカードを登録]                      [使い方を見る]                          |
+----------------------------------------------------------------------------------+

2) 復習対象なし
+----------------------------------------------------------------------------------+
| 今日の復習はありません                                                            |
| 今は新規カードの追加か、カード一覧の整理を進められます                            |
| [学習カードを追加]                          [カード一覧を見る]                    |
+----------------------------------------------------------------------------------+

3) 取得失敗
+----------------------------------------------------------------------------------+
| ! ホーム情報を取得できませんでした                                                |
| ネットワークまたはサーバーの状態を確認してください                                |
| [再試行]                                                                          |
+----------------------------------------------------------------------------------+

モバイル時の意図:
- 学習サマリ -> クイックアクション -> 最近のアクティビティ の順で縦積み
- 主要 CTA は 1 列または 2 列で見切れずに押せる
- 重要指標はカード化して短く表示する

注記:
- 優先プレビュー枠は設けず、情報は学習サマリと最近のアクティビティに集約する
- 学習サマリの固定項目は、今日の復習対象数、期限超過数、未学習数、連続学習日数の 4 指標とする

アクセシビリティ:
- Tab でトップレベルナビゲーション、主要 CTA、補助導線へ順に移動できる
- Enter / Space で主要 CTA を実行できる
- エラー状態では再試行ボタンへすぐ到達できる
