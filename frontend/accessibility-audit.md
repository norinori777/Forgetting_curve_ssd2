# Accessibility Audit (MVP)

対象: カード一覧（検索/フィルタ/選択/復習開始/バルク操作/削除確認モーダル）

## キーボード操作

- 検索入力: Tabでフォーカス可能、入力可能
- ステータスフィルタ: 単一選択のセレクトとして Tab / Arrow で操作可能
- 共用フィルタモーダル: ボタンで開き、ラジオ切替・候補検索・チェックボックス複数選択をキーボードで操作可能
- 一覧の選択: 各カードのチェックボックスにTabで到達し、Spaceで切替可能
- バルク操作: 選択0件のとき操作ボタンは無効（disabled）
- 削除確認モーダル:
  - `role="dialog"` + `aria-modal="true"`
  - Escでキャンセル可能
  - 初期フォーカスは「削除を確定」ボタン

## 目視確認ポイント

- エラー表示が `role="alert"` で通知される
- 空状態で条件リセットができる
- ソート UI が一覧ヘッダ内に埋め込まれ、現在値が視認できる
- タグ/コレクションの選択結果がボタン横サマリに反映される
